# Мовна рамка докторської

Основна мова дисертації: **українська**.

## 1. Загальне правило

Текст має звучати як українська наукова робота, а не як переклад англомовного технічного опису. Англійські позначення використовуються лише тоді, коли:

- це стандартна назва протоколу, алгоритму або бібліотеки;
- український переклад зробив би текст менш точним;
- термін буде використано в таблицях, коді, командах або формальних припущеннях.

## 3. Терміни, які пишемо українською

| Не бажано | Бажано |
|---|---|
| plaintext | відкритий текст |
| ciphertext | шифротекст |
| password guessing | перебір пароля |
| offline dictionary attack | перебір словника поза взаємодією із сервером |
| threat model | модель загроз |
| claim boundary | межа твердження |
| proof boundary | межа доведення |
| metadata | метадані або службові дані |
| messenger protocol | протокол обміну повідомленнями |
| AURA Runtime / safety runtime | середовище виконання політик контекстної безпеки |
| generic classifier | універсальний класифікатор |
| dataset governance | керування наборами даних |
| audit record | аудиторський запис або приватний аудиторський запис |
| server-blind relay | серверно-сліпий канал |
| supervision relay | канал нагляду |
| guardian endpoint | пристрій опікуна |
| endpoint | кінцевий пристрій або кінцева сторона |
| risk signal | сигнал ризику |
| policy-aware decision | рішення з урахуванням політик |
| release-gated evaluation | оцінювання з контрольованим допуском до випуску |

## 4. Перша згадка терміна

Для ключових термінів використовуємо форму:

> перебір словника поза взаємодією із сервером

або:

> серверно-сліпий канал нагляду

Після першої згадки використовуємо український варіант, якщо не йдеться про назву протоколу, репозиторію або поля в коді.

## 5. Стиль формулювання наукової новизни

Використовуємо класичні формули:

- **вперше запропоновано**;
- **вперше розроблено**;
- **удосконалено**;
- **дістало подальшого розвитку**;
- **на відміну від наявних підходів**;
- **це дозволяє**;
- **практична цінність полягає в тому, що**.

Не використовуємо рекламні формули:

- унікальний у світі;
- революційний;
- повністю безпечний;
- гарантує захист;
- доводить безпеку всієї системи.

## 6. Межі тверджень

У кожному розділі потрібно явно розділяти:

- що доведено формально;
- що підтверджено тестами;
- що оцінено експериментально;
- що є інженерною реалізацією;
- що залишається обмеженням або напрямом подальших досліджень.

Це особливо важливо для постквантових тверджень, середовища виконання політик AURA та серверно-сліпого каналу нагляду.
