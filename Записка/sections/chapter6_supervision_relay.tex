\chapter{Метод серверно-сліпого каналу нагляду}

У розділі буде подано метод server-blind supervision relay для передавання safety-сигналів між child endpoint та guardian endpoint без розкриття відкритого тексту серверу, з урахуванням consent, parent-child binding, pair-key provisioning, ACK, revoke та purge lifecycle.
